### 실행법

> 1. `yarn`으로 패키지 설치
> 2. `json-server --watch db.json --port 4000` 으로 JSON server 실행
> 3. `node server.js`로 express 서버 실행
> 4. `yarn start`로 클라이언트 실행


# GongGongPlay

## 🖥️ 프로젝트 소개

React에 다양한 API들로 서울시의 체육시설 공공서비스 예약안내 사이트입니다.
